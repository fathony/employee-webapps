# Employee Web Apps

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 14.x.x.

## Requirement

- nodejs >= 16
- npm >= 8

## Installation

- run `npm install`

## Running the app

- run `npm run start`

## Build Production

- run `npm run build`. The build artifacts
  will be
  stored in the `dist/` directory.
